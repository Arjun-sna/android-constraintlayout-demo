# android-constraintlayout-demo
Demo usage various ConstraintLayout features like
- Add a constraint
- Order position
- Alignment
- Baseline alignment
- Constrain to a guideline
- Adjust the constraint bias
- Set size as a ratio
- Control linear groups with a chain
